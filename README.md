# MLB Live Scores for Pebble

Live MLB scores on your wrist, updated every minute from the official MLB Stats API.

## Features

**Live game** — score, inning, top/bot, balls/strikes/outs, current batter, pitch speed & type, base runner diamond, last play result

**Pre-game** (shown 2 hrs before first pitch) — start time, weather, probable starters with W-L records, TV network

**Final** — W/L/SV pitcher decisions, next scheduled game

**Ticker** — scrolling strip of every other game today; flick your wrist to browse

**Notifications** — vibration pulse when your team scores; Charge fanfare through the speaker when your team homers (Pebble Time 2 / Round 2)

**Always visible** — time, date, optional battery bar, team colors (Pebble Time 2)

## Settings

| Setting | Options |
|---|---|
| Team | All 30 MLB teams |
| Vibrate on Score | On / Off |
| Home Run Sound | Off / Low / Medium / High |
| Test HR Sound | Play Charge ▶ |
| Battery Bar | On / Off |
| Timezone | ET / CT / MT / PT / GMT / CET |
| Ticker Speed | 5s / 10s / 30s / 60s |

## Data

Official MLB Stats API — free, no API key. Refreshes every minute.
